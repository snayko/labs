Hash Cracking Challenge - Advanced

Мета:
Відновити пароль за його хешем.
Перевірити, наскільки стійкою чи нестійкою може бути хеш-функція.
Встановити доцільність і область застосування зазначеної хеш-функції.
Описати методи та алгоритми злому або перебору паролів. Зазначити часові витрати процесу.
Довести на прикладі доцільність використання стійких паролів.
Познайомитись на практиці з алгоритмом хешування з сіллю (і кількістю ітерацій).
Після отримання результату надіслати відновлений пароль на перевірку викладачу.

Завдання:
Вам вдалося отримати дамп бази даних та хеш пароля, який використовується для входу в систему.

 Як виглядає це в базі:

Відомо що використовується формат PBKDF2-HMAC-SHA256.
Також відомо що пароль не є складним - до 9 символів максимум з використанням лише латиниці, цифр (0-9) та спеціальних символів (?, !, #, $, %)

Корисний скрипт для конвертування PBKDF2-HMAC-SHA256 в HashCat readable формат

                  
                    # usage: ./identity-to-hashcat.sh AGKcecrgyBNHkgjKqDlQPW7YlimD+VXPfpAx+gLSR0n1cOdIxFHaxKEF8SgAoaEtqQ==
                    # the hash can be version 2 or version 3
              
                    input=$1
              
                    kind=$(echo $1 | base64 -d | xxd -p -l 1 -s 0)
              
                    if [ 01 == $kind ]
                    then
                      salt=$(echo $1 | base64 -d | xxd -p -c 32 -l 16 -s $((1+4+4+4)) | xxd -r -p | base64 | tr -d '\n')
                      subkey=$(echo $1 | base64 -d | xxd -p -c 32 -l 32 -s $((1+4+4+4+16)) | xxd -r -p | base64 | tr -d '\n')
                      iter=$(echo $1 | base64 -d | xxd -p -g 1 -l 4 -s $((1+4)))
                      hash=sha256
                    elif [ 00 == $kind ]
                    then
                      salt=$(echo $1 | base64 -d | xxd -p -c 32 -l 16 -s $((1)) | xxd -r -p | bbase64 | tr -d '\n')
                      subkey=$(echo $1 | base64 -d | xxd -p -c 32 -l 32 -s $((1+16)) | xxd -r -p | base64 | tr -d '\n')
                      iter=3e8
                      hash=sha1
                    else
                      echo Unsupported hash, exiting.
                      exit
                    fi
              
                    echo "$hash:$(printf '%d' 0x$iter):$salt:$subkey"
